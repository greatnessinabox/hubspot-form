# Contributing

Thank you for your interest in contributing to `@greatnessinabox/hubspot-form`! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/greatnessinabox/hubspot-form.git`
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

```bash
# Install dependencies
pnpm install

# Build the package
pnpm build

# Watch mode for development
pnpm dev

# Type check
pnpm type-check
```

## Making Changes

- Follow the existing code style
- Add tests for new features
- Update documentation as needed
- Ensure TypeScript types are correct

## Submitting Changes

1. Commit your changes with clear messages
2. Push to your fork
3. Create a Pull Request with:
   - Description of changes
   - Why the changes are needed
   - Any breaking changes

## Code Style

- Use TypeScript
- Follow existing patterns
- Write clear, self-documenting code
- Add comments for complex logic

## Reporting Issues

When reporting issues, please include:

- Package version
- Framework/environment details
- Steps to reproduce
- Expected vs actual behavior
- Any relevant error messages

## Questions?

Feel free to open an issue for questions or discussions!
